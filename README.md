# DetailsOrder-dashboard
1. Project Title / Headline
        Build a dashboard that visualizes sales data for an e-commerce company. You can use Power BI to track sales, revenue, and customer demographics across various regions and products.

2. Short Description / Purpose
        DetailsOrder Dashboard is a visually engaging and analytical Power BI report designed to help users explore and compare Profit across state,category. The project focused on analyzing and monitoring key metrics such as profit, order volume, customer demographics, product performance, and sales trends. By leveraging data from the e-commerce platform, the dashboards enabled stakeholders to make data-driven decisions, identify growth opportunities, and track the effectiveness of marketing campaigns.
